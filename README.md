# blog

#### 介绍
用go语言开发的小博客（使用了gin框架和gorm框架）



#### 软件架构
此博客功能包括：

（1）实现了博客的增删查改功能（完善中）

（2）支持修改博客背景logo

（3）实现了登陆注册（jwt-cookie正在完善中）

（4）集成了阿里云oss存储，将博客背景logo存储在oss中

（5）采用了redis缓存策略（redigo），将logo的url缓存在redis中

（6）整体采用了web端的gin框架和持久层的gorm框架

（7）集成了bilibili开源项目flv.js。页面支持播放http-flv直播流
