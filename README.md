# blog

#### 介绍
用go语言开发的小博客（使用了gin框架和gorm框架）



#### 软件架构
此博客功能包括：

（1）博客首页的背景图支持上传到阿里云oss

（2）实现了oss路径缓存在redis中

（3）集成了gin框架

（4）集成了dplay实现播放直播流

（5）集成了链接数据库的gorm框架


博客在持续开发中...

博主也是从java转go方向，刚接触go语言不久。
